# Milestone 3

This milestone establishes the visual and navigation framework for the application.

## Validation checklist

- Dashboard renders KPI cards and two charts.
- Sidebar links navigate to all core modules.
- Each list page initializes a DataTable.
- CSV export works on list pages.
- Sidebar collapses on desktop and slides on mobile.
- All displayed data comes from `app/common/fake_data.py`.
